About This Project

This is a quilt project for ENGL 1102 in Georgia Tech. Our project is a quilt that stitched together scenes from Frankenstein. We chose key scenes from the novel and then picked out images that represented each one.

This specific website is created to provides interactive interface to our quilt for everyone.
